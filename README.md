## Requirements

- Python 3.x
- tkinter (usually comes with Python)
- pygame
- PIL (Pillow)
- speech_recognition (optional, for speech features)

## Installation

1. Clone this repository:
```bash
git clone https://github.com/YOUR_USERNAME/music-player.git
cd music-player
```

2. Install required dependencies:
```bash
pip install pygame pillow speechrecognition
```

## Usage

1. Run the application:
```bash
python musicplayer.py
```

2. Click the "Add Songs" button to add MP3 files to your playlist
3. Select a song from the list and click Play
4. Use the control buttons to play, pause, stop, skip to next, or go to previous song
